Portfolio Reflection

Q1: How is it coming along thus far?
The portfolio is built as a single-page website with sections for About Me, Experience, Activities, Education, and Skills. The content is organized and the layout is functional.

Q2: Are you happy with your page layouts, ease of navigation, and color scheme?
The page uses a clean card-based layout with anchor link navigation. The color scheme combines navy (#1a365d) and blue (#3182ce), creating a professional appearance. Navigation allows visitors to jump between sections.

Q3: Are the fonts clear and easy to read?
The fonts used are Segoe UI, Roboto, Helvetica, and Arial. These are standard system fonts that render clearly across devices.

Q4: What have you learned through the process of creating this digital portfolio?
The process of creating this portfolio showed me how to organize technical experience into concise, scannable content while still conveying the depth of my achievements in cybersecurity competitions and technical roles.

Q7: What resources might you use to assist you?
The resources that helped guide my design decisions include W3Schools for HTML and CSS reference, MDN Web Docs for documentation, and my course's Digital Portfolio Resources page for best practices.
